## 备选方式：服务器版（需要 FFmpeg）

如果你想使用服务器端转换（速度更快），需要先安装 FFmpeg。

### macOS 15 安装 FFmpeg 的方法：

**方法 1: 使用 Conda（你已经有 conda）**
```bash
conda install -c conda-forge ffmpeg -y
```

**方法 2: 手动下载**
1. 访问: https://evermeet.cx/ffmpeg/
2. 下载 ffmpeg
3. 解压并移动到系统路径

安装完成后访问: http://localhost:3000

---

## 📝 使用步骤

1. 打开浏览器访问上面的链接
2. 拖放或选择 WebM 文件
3. 点击"开始转换"
4. 等待转换完成
5. 下载 MP4 文件

就这么简单！
